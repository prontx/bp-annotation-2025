# Annotation User Interface 
Author: Jakub DugoviÄ

## Prerequisites
- Node.js: ^20.12.0
- yarn: ^1.22.21

## Run Dev Server
install dependencies (only the first time)
```
yarn -i
```

run dev server
```
yarn dev
```

*job_id is loaded from the URL param, append ?job_id=JOD_ID, where JOB_ID is your test job id.*
